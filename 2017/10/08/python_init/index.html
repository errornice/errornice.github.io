Pythone系列(1)之Linux 安装Python

发表于 2017-10-08 | 分类于 Pythone | | 阅读次数

环境: centos 6.7

安装Python

  1. 下载python的安装包
    使用wget命令下载python 的安装包,本文安装的是python2.7.3
    wget https://www.python.org/ftp/python/2.7.3/Python-2.7.3.tar.bz2
    如果需要安装其他版本在下面的地址中选择需要下载的版本
    https://www.python.org/ftp/python/

2.解压安装包

  • tar -jxvf Python-2.7.3.tar.bz2

3.安装

  • 进入解压目录
    cd Python-2.7.3
  • 执行./configure，主要的作用是对即将安装的软件进行配置，检查当前的环境是否满足要安装软件的依赖关系。
    ./configure
  • 如果出现下面的异常说明没有安装gcc

    则需要安装gcc
  • 安装gcc
    yum install -y gcc
    安装完成后再执行./configure
    没出现异常情况就继续,使用make命令进行编译
    make (或者使用make -jN N 表示CPU的核心数 例如 make -j2 可以提高编译效率 )
    编译完成后进行安装
    make install
