HTML标签

发表于 2016-07-08 | 分类于 web前端 | | 阅读次数

上一篇博客介绍了什么叫HTML，那么接下来我们就开始学习HTML的标签。HTML就是负责描述页面的语义的，所以HTML并不能让文字居中，不能更改文字字号、颜色，因为这都是属于样式范畴。HTML只能做一件事情,就是通过标签对，给文本增加语义。

HTML标签

h标签

<h1>到<h6> 都是标题标签:

h标签没有嵌套关系的。由于h3跟着一个h2，所以，我们自己就知道了这个h3是h2子标题。
h是容器级的标签。理论上里面可以设置p、ul,但是在语义上不要这么写。

p标签

段落,英语是paragraph “段落”的缩写。
html标签是分等级的,html将所有的标签分为两种:容器级和文本级
顾名思义,容器级标签,里面可以放置任何东西;文本级的标签里面,只能放文字、图片、表单元素。
p标签是一个文本级标签,所以p里面只能放文字、图片、表单元素。其他的一律不能放。
试着把h放到p到里：

浏览器不允许你这么做，我们使用Chrome的F12审查，你会发现，浏览器自己把p封闭掉了，不让你去包裹h1。

img图片标签

页面上可以插入图片,能够插入的图片类型:jpg(jpeg)、gif、png、bmp。

img标签,可以在页面上插入图片。
img是英语image”图片”的缩写,src就是”source”的缩写
src是img标签的属性,baby.jpg是这个属性的值。
img是自封闭标签,也称为单标签。

为什么是单标签？

因为如果不是但标签,那么标签之间可以放东西,但是这样的话就给img标签增加了语义,所以会有单标签。

alt属性是alternate”替代”的意思,就是图片标签里面的图片无法显示了,就会出现替代的文字

相对路径与绝对路径

绝对路径：文件在硬盘上的具体位置。
相对路径：文件相对于引入者的位置。

a标签

一个网站是由很多html网页组成的,html网页之间能够通过超级链接相互跳转。

a是英语anchor”锚”的意思,就好像这个页面往另外一个页面扔出了一个锚.是一个文本级的标签。
href是英语hypertext reference超文本地址的缩写。
属性：

  • href 设置链接路径 
    访问内网：相对路径，绝对路径
访问外网：需要加上http协议。 例如：http://www.baidu.com
  • name 设置锚点，相当于书签，配合herf使用，设置锚点，a标签可以没有内容。
  • target 定位资源打开位置，一般可以配合框架使用。

列表标签

无序列表

无序列表,用来表示一个列表的语义,每个列表项之间是不分先后的。
ul:就是英语unordered list ，”无序列表”的意思
li:就是英语list item ,”列表项”的意思。
这是一组标签,也就是说li不能单独存在,必须包裹在ul里面。还有就是ul里面只能有li标签,不允许放其他标签(放也不会报错,但是语义读不通)

浏览器会默认给无序列表小圆点,这个叫”先导符号”。li是一个容器级标签,li里面什么都能放。

有序列表

ordered list 有序列表,用ol表示。

dt、dd都是容器级标签,想放什么都可以。所以用什么标签得看:
用什么标签,不是根据样子决定的,而是根据语义。

div和span

div的语义是division”分割”;span的语义是”范围、跨度”。
div在浏览器中,默认是不会增加任何样式的,但是加上这个标签语义就变了,div中所有的元素是一个小区域。
div标签是一个容器级标签,里面什么都能放,也可以在里面放div。

span也是表达”小区域、小跨度”的标签,但是span是一个”文本级”的标签。
span里面是放置小元素的,div里面是防止大东西的。

div+css :div标签负责布局,负责结构,负责分块。css负责页面样式。

表单

表单就是收集用户信息,让用户填写的。

form 是英语表单的意思,form标签里面有action属性和method属性，action属性就是表示表单提交到哪里,method属性表示用什么http方式提交(get、post)。

input标签

input 表示”输入”,指的是这是一个输入框。
type 表示”类型”,类型有：

>* 文本框 text。输入的文本信息直接显示在框中。
>* 密码框 password。输入的文本以原点或者星号的形式显示。非明文
>* 单选框 radio 如：性别选择。
>* 复选框 checkbox 如：兴趣选择。
>* 提交按钮 submit 用于提交表单中的内容。
>* 重置按钮 reset 将表单中填写的内容设置为初始值
>* 附件框 file 后期扩展内容，会自动生成一个文本框，和一个浏览按钮。例如：照片
>* 隐藏字段 hidden 在页面上不显示，但在提交的时候随其他内容一起提交。例如：用户编号
>* 按钮   button 可以为其自定义事件。
>* 图片提交按钮 image 是一个图片形式的提交按钮。用来美化提交按钮的。提交表单数据时，会将鼠标点击图片的坐标一起封装。较少使用。

text 表示”文本”
value 表示值,value属性就是默认值
input 是一个自闭标签,我们到现在学习了的自闭标签如下:

选择框

select，定义了一个选择框
option，定义了一个选择框条目

属性：

>* name    用来指定选择项的名称。即参数名称
>* value    用来指定选择项的值。即参数值
>* selected    用来设置选择框的默认选中。值为selected为默认选中

文本域

textarea，定义一个文本域输入框

属性：

>* name    用来指定文本域的名称，即参数名
>* cols    定义文本域显示几列
>* rows    定义文本域显示几行

和<input type=“text”/>区别：
1、文本域可以换行，而文本框不可以
2、文本域的值是写在内容体中的，文本框的值是在value中

input select textarea 统称为表单项（表单输入项）

label标签

在单选按钮中,男、女这两个汉字,和input 元素没有关系。

label就可以解决这个问题,让文字和单选按钮关联。

框架标签

作用：将网页分割成几个部分，在每个部分分别显示不同内容。
注：框架标签不可以放到里，一般为了代码的可读性，会到和之间。或者不要body也可以。
frameset
表示框架标记，用以宣告HTML文件为框架模式，并设定视窗如何分割。
常用属性：

>* cols    垂直切割。接收整数值或百分比。*代表占用剩余空间
>* rows    横向切割。接收整数值或百分比。*代表占用剩余空间 吗

frame
表示框架页面，用来承载页面，并修改参数属性值
常用属性：

>* src    表示资源的路径
>* name    设置框架的名称
